Math Decoder is a cryptography-focused mathematics worksheet aimed at helping young children have fun while practicing mathematics. Parents are prompted to enter a secret message plus an optional additional message. Using the jQuery javascript library, this web application will transform this secret message string into a series of mathematical expressions that need to be solved. With each expression and accompanying letter corresponding to a blank solution provided below, it is up to the decoder to solve the expressions to decrypt the message. This web app was designed to be as printable as possible. Users can input any arbitrary sentence, and the program will do all the work generating the expressions. Simply send it to your printer and let your child decode the message.

Chore Chart is a proof of concept web application that is currently non-functional and currently in development. This is a chore tracking solution for parents and an interactive method for children to find new ways to enjoy working on their often boring chores. Similar to Math Decoder, it was based on this concept we previously made on Microsoft Word for Lillian. We laminated a chore chart and she marked off her tasks. My goal was to make it more interactive, fun, and engaging for her. I wanted a measurable solution for parents to track the progress and keep records of her actions, and thus Chore Chart was born. Please understand: The website is front-end only and is strictly eye-candy. No actual logging is taking place and no back-end technology has been implement. That will come in due time. This is just a proof of concept.
